# TaskFlow – To-Do Web Application

TaskFlow is a modern and interactive To-Do web application designed to help users manage their daily tasks efficiently.  
It allows users to create tasks, organize them into lists, set priorities, track completion, and schedule tasks with date and time.

## Features

- Add new tasks easily
- Edit or delete tasks
- Mark tasks as completed
- Set priority levels (High, Medium, Low)
- Assign tasks to custom lists
- Set date and time for tasks
- Search and filter tasks
- Task statistics and progress tracking
- Responsive and modern user interface
- Data stored using browser LocalStorage

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage for saving tasks

## Functionalities

- Task creation and management
- Priority tagging
- List organization
- Task filtering and searching
- Progress tracking and statistics
- Task editing with modal interface
